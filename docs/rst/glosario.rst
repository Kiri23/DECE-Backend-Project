.. _glosario:
Glosario
========

.. glossary::

    Class Based Views
      Te ahorran muchos el trabajo. Porque la mayoria ya esta hecho.

    QuerySet
      Termino que se utiliza para referirse al query que se hace a la base de datos. Tambien es una clase en Django

    ORM
      Los metodos que contiene el SQL que se va a ejecutar en la base de datos. El ORM solamente contiene metodos como ``filter`` y ``get``. Dentro de esos metodos es que se Django hace el SQL. Usted como programador no se tiene que preocupar por saber SQL porque el ORM hace esto por usted.
